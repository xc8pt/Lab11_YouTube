## Описание
В этом проекте мы реализовывали с помощью html и css языков, клон YouTube(запрещённый в РФ). Даже добавили видео в виде заглушек.
## Реализованные функции
 - [] Адаптивный хедер с поиском
 - [] Боковая панель навигации
 - [] Категории (чипсы) с интерактивностью
 - [] Сетка видео с карточками
 - [] Hover-эффекты на карточках
 - [] Полная адаптивность под все устройства
---
## Технологии
- HTML5
- CSS3
- Flexbox
- CSS Grid
